Wellframe code challenge:

1. Created an authorization system that allows users to login or create a new user. User can't access site unless logged in, when making new user must select role, patient or caregiver

2. When logged in, the user is directed to an index page of all conversations with caregivers showing the conversation, who initiated, the subject as well as count of messages vs un read

3. User clicks on conversation and it shows the message thread, the user can click reply to add to it or mark read to make the message read.

4. User can go to paths and get json or create messages,mark as read via http verbs.
